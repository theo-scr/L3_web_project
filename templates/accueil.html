Accueil

Bienvenue sur notre site.

Vous trouverez ci-dessous la liste des différentes actions que vous pouvez réaliser sur notre site.

Que souhaitez-vous faire ?

  • C'est votre première visite sur notre site ? -> Rendez-vous dans l'onglet "Règles du jeu" (dans le sous-menu "Aide") pour tout connaître sur le Pixel Factory.
  • Impatient de commencer une nouvelle partie ? -> Cliquez sur l'onglet "Jouer une partie". N'attendez plus.
  • Vous souhaitez connaître votre niveau et vous situez par rapport aux meilleurs joueurs ? -> Vous trouverez dans l'onglet "Highscores" votre bonheur.
  • Votre dernière partie était mémorable ? -> Grâce à l'onglet "Revoir une partie" vous pourrez la revoir autant de fois que vous le souhaitez.
  • Une question ? Besoin d'aide ? -> Contactez-nous via l'onglet "Contact"
  • Vous souhaitez vous déconnecter ? -> Rendez-vous un peu plus bas.

Déconnexion

En espérant vous revoir bientôt !

A propos

Projet L3 SID.
L'objectif était de réaliser un site web en manipulant des données stockées dans une base de données Oracle.
Sujet: PIXEL Factory.
Outils:
  • SGBD : Oracle
  • Serveur Python Flask
  • Langage côté serveur: Python - API Flask
  • Langage côté client: Javascript (ES5), HTML 5, CSS
  • Librairie Javascript: JQuery
  • Navigateur: Firefox
