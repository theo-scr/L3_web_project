Création d’un site web à partir d’une base de données permettant de jouer au jeu du Pixel Factory

Installation: 
	1) Ouvrir l'application Jupyter
	Attention : il faut exécuter la commande suivante dans la console qui démarre Jupyter:
		export LD_LIBRARY_PATH=/opt/oracle/instantclient_19_5:$LD_LIBRARY_PATH
	
	2) Ouvrir le fichier Notebook "server.ipynb".
	
	3) Exécuter toutes les cellules et cliquer sur le lien qui apparaît après l'exécution de la dernière.




Contenu:
	- Le dossier 'templates' contient tous les fichiers HTML
	- Le dossier 'static' contient 3 sous-dossiers:
		- 'css' qui contient le fichier 'main.css' permettant la mise en page du site
		- 'img' qui contient les cinq images/icons et le logo du site
		- 'js' qui contient les deux fichiers JavaScript permettant de rendre certaines pages dynamiques.

	- Le dossier 'database' contient tous les fichiers permettant de créer la base, insérer les données, créer des procédures, triggers, etc 
	- Le fichier 'server.ipynb' permettant de mettre en route le site
	- Le fichier 'rapport.pdf' contenant le rapport détaillé du projet
	- Le fichier 'video.txt' contenant le lien vers votre video de démonstration.
	- Le fichier 'README.txt' permettant de présenter les différents dossiers et fichiers du projet
